## Sobre o projeto

O NovaMind é um hub de conteúdo independente focado em IA e Cloud Computing para o público técnico brasileiro. O site reúne artigos em tempo real, ferramentas curadas e cursos em alta — tudo sem patrocínio disfarçado.

**Desenvolvido por [Pedro Peterson](https://www.linkedin.com/in/pedro-peterson-260850408) — PetersonCodeDev**

---

## Funcionalidades

- 📰 **Artigos em tempo real** via NewsAPI — atualização automática a cada 30 minutos
- 🛠️ **Ferramentas curadas** — seleção manual das melhores ferramentas de IA e Cloud
- 🎓 **Cursos em alta** — links diretos para cursos reais incluindo a Anthropic Academy oficial
- 📬 **Formulário de contato** funcional via Web3Forms
- 📱 **Responsivo** — funciona em desktop, tablet e mobile
- ⚡ **Zero dependências** — HTML, CSS e JavaScript puro, sem frameworks

---

## Tecnologias utilizadas

- HTML5 / CSS3 / JavaScript vanilla
- [NewsAPI](https://newsapi.org) — artigos em tempo real
- [Web3Forms](https://web3forms.com) — formulário de contato
- [Google Fonts](https://fonts.google.com) — Outfit + JetBrains Mono
- GitHub Pages — hospedagem gratuita
